Uber Eats • Overview

Building a tool for ops teams to manage events and crises

The opportunity

Ops team needs to create restrictions when there are events or crises. E.g. a parade or police street closure. The existing experience was manual and restrictive. Some walls were made in a legacy too. Some, you needed to run scripts. Walls weren’t flexible - blocked everything. E.g. No way to allow only bikes.

Designing and scoping a new tool

Looping ops users into the design review process. Helped reprioritize requirements / uncover new ones. Initial product requirements thought viewing walls in a map was important but instead more important to improve wall richness and enable bulk creation. Wall creation flow Deep dive into wall types / restrictions. Technical challenges. How to communicate different types of restrictions Viewing and editing walls Bulk wall creation
